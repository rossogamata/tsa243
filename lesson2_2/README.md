## Мета заняття

Опанувати інструменти: - управління процесами - створення архівів різних
типів - зміни файлових прав доступу - створення жорстких та символічних
посилань

------------------------------------------------------------------------

# Частина 1. Управління процесами

## Завдання 1. Аналіз процесів

1.  Вивести список процесів:

    ``` bash
    ps aux
    ```

2.  Відобразити дерево процесів:

    ``` bash
    ps -ejH
    ```

3.  Запустити моніторинг:

    ``` bash
    top
    ```

4.  (Опціонально)

    ``` bash
    sudo apt install htop
    htop
    ```

------------------------------------------------------------------------

## Завдання 2. Керування процесами

1.  Запустити процес у фоні:

    ``` bash
    sleep 300 &
    ```

2.  Знайти PID процесу:

    ``` bash
    ps aux | grep sleep
    ```

3.  Завершити процес:

    ``` bash
    kill PID
    ```

4.  Примусове завершення:

    ``` bash
    kill -9 PID
    ```

------------------------------------------------------------------------

# Частина 2. Архівування

## Підготовка

``` bash
mkdir -p project
cd project
echo "File1" > file1.txt
echo "File2" > file2.txt
echo "File3" > file3.txt
cd ..
```

## Створення tar.gz архіву

``` bash
tar -czvf project.tar.gz project
```

## Перевірка вмісту архіву

``` bash
tar -tzf project.tar.gz
```

------------------------------------------------------------------------

# Частина 3. Права доступу

## Встановити права:

-   власник -- повний доступ
-   група -- лише читання
-   інші -- без доступу

``` bash
chmod 740 project/*
```

Перевірка:

``` bash
ls -l project
```

------------------------------------------------------------------------

# Частина 4. Посилання

## Створити символічне посилання на архів:

``` bash
ln -s project.tar.gz project_link
```

Перевірка:

``` bash
ls -l
```

------------------------------------------------------------------------

# Підсумкове завдання

1.  Створити каталог `project`

3.  Встановити права 740
